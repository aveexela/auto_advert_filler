\# Multimodal Ads Parsing Project



\## Постановка задачи

Цель проекта — автоматическое извлечение и автозаполнение информации

о товарах по данным онлайн-объявлений (HTML и изображения).

Проект ориентирован на мультимодальный ML-пайплайн.



\## Текущий статус

На текущем этапе реализован:

\- парсинг HTML-страниц объявлений;

\- извлечение заголовка, категории, характеристик и описания;

\- базовая обработка изображений.



Вся реализация на данном этапе представлена в виде Jupyter Notebook.



\## Структура репозитория

\- `notebooks/parser.ipynb` — основной код парсинга

\- `data/sample\_html` — пример HTML-страницы объявления

pip install -r requirements.txt
